First-Grade Admission Assessment App

Overview

This project is a simple single-page arithmetic assessment application designed for first-grade admission screening.

The application evaluates basic numeracy skills using three arithmetic questions:

* Addition
* Subtraction
* Multiplication

Students must answer all three questions correctly to pass the assessment.

Features

* Randomized single-digit arithmetic questions
* Addition, subtraction, and multiplication operations
* Input fields for student answers
* Automatic evaluation system
* Pass/Fail result display
* Child-friendly single-page interface


Technologies Used

* HTML
* CSS
* JavaScript

Project Files

* `index.html` – Main structure of the application
* `style.css` – Styling and layout
* `app.js` / `test_logic.js` – Question generation and evaluation logic

Evaluation Logic

* 3/3 correct answers → Pass
* Less than 3 correct answers → Fail

Result Messages
1.Pass
"Congratulations! You passed!"

2.Fail
"Thank you for trying! Please see the administrator."

Scope

This project focuses on:

* Single-page UI
* Client-side evaluation
* Basic arithmetic logic
* Simple and accessible user experience
